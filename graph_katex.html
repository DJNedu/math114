그래프 풀이 — \(f = \bigl(y-(2x+3)\bigr)^{2} + (y+2)^{2} - 6\)

두 직선까지 \(y\)거리\(^2\) 합 \(- 6\)  →  간격 \(d\)일 때 최솟값 \(= \dfrac{d^{2}}{2} - 6\)
\(f = \color{#00e5ff}{\bigl(y-(2x+3)\bigr)^{2}} + \color{#ec4899}{(y+2)^{2}} - 6\)
두 축 \(y = 2x+3\)과 \(y = -2\)의 간격 \(d\)일 때, \(y\)는 중점에서 최소
최솟값 \(= \dfrac{d^{2}}{2} - 6\)
\((1)\) \(x,\, y\) 실수

두 축 일치: \(2x+3 = -2\) → \(x = -\tfrac{5}{2},\; y = -2\)

간격 \(d = 0\) → 거리 합 \(= 0\)

\(f\!\left(-\tfrac{5}{2},\,-2\right) = 0 + 0 - 6 =\) \(-6\)
\((2)\) \(x \geq 0,\; y \geq 0\)

\(x \geq 0\)이면 축\(1 = 2x+3 \geq 3\), 축\(2 = -2\) → \(d \geq 5\)

\(x = 0\)일 때 \(d = 5\) 최소, 중점 \(y = \tfrac{1}{2}\)

\(f\!\left(0,\,\tfrac{1}{2}\right) = \tfrac{25}{2} - 6 =\) \(\dfrac{13}{2}\)
\((3)\) \(x,\, y\) 정수

축\(1 = 2x+3\) (홀수) \(\neq -2\) → 일치 불가

\(d\) 최소 \(= 1\): \(x = -2\) (축\(1 = -1\)) 또는 \(x = -3\) (축\(1 = -3\))

중점 \(y = -\tfrac{3}{2}\) (비정수) → 위아래 정수에서 동일

\(f(-2,-1) = f(-2,-2) = f(-3,-2) = f(-3,-3) =\) \(-5\)
최솟값 정리
\((1)\) \(-6\)    \((2)\) \(\frac{13}{2}\)    \((3)\) \(-5\)
